Server Execution Documentation

Assignment Requirement: Execute the website using Apache HTTP Server.

Below are the exact steps I followed to configure and run this site on my local Apache server:

  1. Installed XAMPP: Downloaded and installed XAMPP for Windows to set up a local Apache HTTP environment.
  2. Configured htdocs: Navigated to the Apache document root folder at C:\xampp\htdocs\.
  3. Deployed Files: Created a new directory named assignment7 inside htdocs and copied index.html, about.html, contact.html, and style.css into it.
  4. Started Apache: Opened the XAMPP Control Panel and started the Apache module (running on default port 80).
  5. Execution: Accessed the website via my web browser by navigating to http://localhost/assignment7/index.html.
